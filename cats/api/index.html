ROBOS

This API is powered by another API called robohash.org (Thanks robohash.org I luv U!)

So lets call this a double-API.

How to use:

- Get a random cat

You can get the amount of cats at vicwolrocket.github.io/robos/assets/repeat.txt

Then redirect the user to vicwolrocket.github.io/robos/assets/[pick random, 1, amount of cats].png

- Get a specific cat

Go to vicwolrocket.github.io/robos/assets/ID.png
